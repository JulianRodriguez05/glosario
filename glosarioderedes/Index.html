Mi glosario personal de Sistemas Operativos

Por: Julián David Rodriguez Orjuela



A continuación encontrarás las definiciones principales de el glosario:



Área dinámica

Experimental trata temas relacionados con las acciones dinámicas y sus efectos sobre sistemas estructurales y mecánicos, en relación con el estudio de vibraciones.



Contact Center

Un contact center es un centro de atención al cliente donde se gestionan y atienden las comunicaciones de los clientes



DFS (Sistema de Archivos Distribuido):

Un sistema de archivos que se extiende a través de múltiples servidores y puede proporcionar redundancia y tolerancia a fallos.



Disco Duro

Dispositivo de almacenamiento de datos, que gracias a un sistema de grabación magnética es capaz de almacenar archivos digitales.



DNS

Sistema de nombres de dominio



Ethernet

Estándar de comunicación de red por cable más comúnmente utilizado.



Fibra óptica

Hilo muy fino de material transparente empleado en redes de datos y telecomunicaciones como medio de transmisión.



Firmware

Software de bajo nivel almacenado en dispositivos hardware, como BIOS o firmware de dispositivos periféricos, que controla su funcionamiento básico y proporciona interfaces para el sistema operativo.



FTP (Protocolo de Transferencia de Archivos)

Protocolo utilizado para transferir archivos entre sistemas a través de una red, como Internet.



Interfaz de Usuario (UI)

La parte del sistema operativo que permite la interacción entre el usuario y el sistema, puede ser gráfica (GUI) o de línea de comandos (CLI).



IP

Significa Protocolo de internet



ISP (Proveedor de Servicios de Internet):

Empresa que proporciona acceso a Internet y otros servicios relacionados, como la conexión de red y el alojamiento web.



Modem:

Dispositivo que convierte señales digitales en analógicas y viceversa, permitiendo la conexión de dispositivos a una red de comunicaciones.



Network File System (Sistema de Archivos en Red):

Sistema que permite acceder y compartir archivos almacenados en servidores remotos a través de una red de área local o internet.



Router:

Dispositivo de red que permite la interconexión entre diferentes redes y facilita el enrutamiento de paquetes de datos.



Sistema de Archivos

La estructura que el sistema operativo utiliza para organizar y almacenar archivos en un dispositivo de almacenamiento.



Sistema de Archivos Distribuido (Distributed File System - DFS)

Un sistema de archivos que se extiende a través de múltiples servidores y puede proporcionar redundancia y tolerancia a fallos. Sistema de Archivos en Red (Network File System - NFS): Protocolo que permite a los usuarios acceder y compartir archivos a través de una red.



Sistema de Archivos en Memoria (Memory File System):

Sistema de archivos que utiliza la memoria RAM en lugar de dispositivos de almacenamiento de disco para almacenar y manipular archivos, lo que puede mejorar el rendimiento en ciertas aplicaciones.



Sistema de Cola (Queueing System):

Mecanismo utilizado por el sistema operativo para administrar la ejecución de procesos en una cola, generalmente basada en prioridades o algoritmos de programación.



Sistema de Tiempo Real (RTOS)

Un sistema operativo diseñado para ejecutar aplicaciones con restricciones de tiempo, donde las respuestas deben ocurrir en tiempos estrictamente definidos.



SMTP

Significa protocolo simple de transferencia de correo



Sistema Operativo (SO)

Un software que gestiona los recursos de hardware y proporciona servicios comunes para los programas de aplicación.



URL (Localizador Uniforme de Recursos):

Dirección web que identifica de manera única un recurso en Internet, como una página web o un archivo.



Virtualización

La capacidad de un sistema operativo para crear entornos virtuales que simulan hardware y software reales.



WAN (Red de Área Amplia):

Red de comunicación que cubre un área geográfica extensa, como un país o continente, mediante la conexión de múltiples redes locales.



WLAN

son un tipo de conexión que utiliza la tecnología de radio en sus distintos estándares ―como el WiFi―, de manera que es posible realizar la conexión a internet sin tener que utilizar cables de por medio.
